# PlantBox QGIS Plugin

## Overview

The PlantBox QGIS Plugin provides QGIS desktop integration with the
[PlantBox database](https://github.com/telecomcraft/plantbox-db-api), and includes toolbars, menus, and
dialogs designed to make creating and managing OSP records easier.

## Requirements

The plugin is developed for QGIS 4, Python 12, and PyQt 6, and requires a PlantBox database installed and reachable by QGIS.
